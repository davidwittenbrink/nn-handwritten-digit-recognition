NeuralNetClassifier

Related Doc: package root

object NeuralNetClassifier

An object containing several pure functions to solve classification tasks using neural networks with Stochastic Gradient Descent as learning algorithm. The implementation sticks to the math as explained in Michael Nielsen's fantastic book about neural networks. For an example how you could use this classifier for optical image recognition on the MNIST data set, see how it is used in the main function of this project.

See also

http://neuralnetworksanddeeplearning.com Main.main

  27. def predictRecursive(weights: List[DenseMatrix[Double]], biases: List[DenseVector[Double]], input: DenseVector[Double], layerIdx: Int, zs: List[DenseVector[Double]], activations: List[DenseVector[Double]]): (DenseVector[Double], List[DenseVector[Double]], List[DenseVector[Double]])

    Given a config and input this function does a feedforward run and returns a tuple of

    Given a config and input this function does a feedforward run and returns a tuple of

    • The network output
    • A list of z vectors calculated along the way
    • A list of activation vectors along the way (the output vectors of each layer)

    The output of the network is calculated in a recursive manner. To do this in an efficient (tail-recursive) way, this function needs some helper arguments. If you don't want to take care of the initialisation of these arguemnts yourself, the function predict has an easier signature and does this for you.

    weights

    A list of the weight matrices of the network (e.g. generated by the function generateRandomNetConfig)

    biases

    A list of the bias vectors of the network (e.g. generated by the function generateRandomNetConfig)

    input

    The input to the network

    layerIdx

    The index of the layer in the current recursion step (0 is input layer)

    zs

    A list of z vectors that is calculated and accumulated along the way

    activations

    A list of activation vectors that is calculated and accumulated along the way

    returns

    Tuple of the network output, list of z vectors, list of activation vectors

    See also

    http://neuralnetworksanddeeplearning.com/chap2.html#warm_up_a_fast_matrix-based_approach_to_computing_the_output_from_a_neural_network

  29. def recursiveCalcDeltas(weights: List[DenseMatrix[Double]], biases: List[DenseVector[Double]], networkOutput: DenseVector[Double], target: SparseVector[Double], zs: List[DenseVector[Double]], layerIdx: Int, deltas: List[DenseVector[Double]]): List[DenseVector[Double]]

    Returns a list of delta vectors needed for gradient descent.

    Returns a list of delta vectors needed for gradient descent.

    The list of delta vectors is calculated in a recursive manner. To achieve this efficiently (tail-recursive), some helper arguments are required. The function calcDeltas initialises these arguments correctly and offers an easier signature.

    weights

    A list of the weight matrices of the network (e.g. generated by the function generateRandomNetConfig)

    biases

    A list of the bias vectors of the network (e.g. generated by the function generateRandomNetConfig)

    networkOutput

    The output of the network for the training sample

    target

    The desired output of the network

    zs

    A list of z vectors that were calculated while calculating the network output

    layerIdx

    The index of the layer in the current recursion step. This should be initialised to the length of weight matrices - 1 (meaning the nr of total layers - 2).

    deltas

    The deltas calculated so far. This list is accumulated during the recursion steps and ultimately returned in the last step.

    returns

    List of delta vectors

    See also

    http://neuralnetworksanddeeplearning.com/chap2.html
